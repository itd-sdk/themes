# themes
Темы для ИТД

## Как поставить

1. Скачивайте расширение "Custom CSS" ([chrome](https://chromewebstore.google.com/detail/custom-css-by-denis/cemphncflepgmgfhcdegkbkekifodacd), [firefox](https://addons.mozilla.org/ru/firefox/addon/qb-custom-css/)) (можно любой другой похожий)
2. Предоставляете ему разрешение на изменение на странице (если попросило)
3. В окно вставляете код из `theme.css` (у тем) или `patch.css` (у патчей) (файл находится внутри папки)
4. Перезагружаете страницу (некоторые расширения не загружают изменения сразу)

## Прочее
В темах нет никаких пизделок ключей, это дефолтный CSS (стили для страниц), если сомневаетесь то можете кинуть код нейронке.

Если хотите добавить свою тему, создавайте PR. Вы должны создать папку, закинуть туда скриншоты вашей темы, добавить README со скриншотами и файл theme.css.
